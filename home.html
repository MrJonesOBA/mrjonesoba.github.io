This page has three tabs at the top, each showing different content.


You'll find all the files you need to make this here:

Clicky Clicky


Right-click the .css and .js files and choose Save link.

Put them into the root folder of your website (where your index page is).

Copy and paste this code into the <head> section:
<link href="tabcontent.css" rel="stylesheet" type="text/css" />
<script src="tabcontent.js" type="text/javascript"></script>

Copy and paste this code into the <div> where you want your tabs:
<ul class="tabs"> <li><a href="#view1">Features</a></li> <li><a href="#view2">How to Use</a></li> <li><a href="#view3">Source Code</a></li> </ul> <div class="tabcontents"> <div id="view1"> content 1 </div> <div id="view2"> content 2 </div> <div id="view3"> content 3 </div> </div>
  • Look at the code in red on the previous tab.
  • Notice how each link corresponds to a div id.
  • This allows the javascript to link each tab to the appropriate content.
  • Use the same naming convention to add more tabs if required.

Clicky Clicky


Right-click the .css and .js files and choose Save link.

Put them into the root folder of your website (where your index page is).


Now open the images folder and save each of the images.

Put them into the images folder of your website.

Copy and paste this code into the <head> section:
<link href="js-Image-slider.css" rel="stylesheet" type="text/css" />
<script src="js-Image-slider.js" type="text/javascript"></script>

Copy and paste this code into the <div> where you want your image slider:
<div id="sliderFrame"> <div id="slider"> <img src="images/image-slider-1.jpg" alt="" /> <img src="images/image-slider-2.jpg" alt="" /> <img src="images/image-slider-3.jpg" alt="" /> <img src="images/image-slider-4.jpg" alt="" /> <img src="images/image-slider-5.jpg" alt="" /> </div> </div> <!-- #sliderFrame -->
  • You need to replace the example images you downloaded with your own.
  • Use Photoshop to resize/crop your images to the required size.
  • Make sure your slider images are in your website's Images folder.
  • Check that your filenames match those in the html
    (e.g. "image-slider-1.jpg").
  • Use the alt="" attribute to add overlay text to each image if required.
  • Edit the js-image-slider.css file:
    • Look for the comments marked ATTENTION.
    • Adjust the dimensions to match those of your images.

  • Get Code & Customise
  1. Head over to Google Maps.
  2. Ensure that the map you'd like to embed appears in the current map display.
  3. Click the settings gear icon in the bottom right.
  4. Click Share and embed map.
  5. In the box that appears, make sure the Embed map tab is selected.
  6. Select a map size (you can edit this in more detail later).
  7. Copy the embed code and paste it into the <div> where you want your map.
  8. The code has width and height properties that you can change if necessary.
  • Example
  • Get Code & Customise
  1. Head over to MailChimp and create an account using your obamail e-mail.
  2. Fill in the form for your company (e.g. Urban Mobile Phones). You'll have to make some of the information up.
  3. Once you're activated and signed in, click the Create a list option.
  4. Fill out the form using your company name again and obamail email.
  5. Save your list and then choose the Signup forms option.
  6. Select Embedded forms.
  7. The Super Slim option provides the simplest looking subscribe form. The Advanced option provides a guide for customising your form using CSS but this is not essential.
  8. Copy the html and paste it into the <div> where you want your form.
  9. As a final step, you should move the css section of the MailChimp code into your external style sheet and remove the <style> tags.
  10. You should also move the Mailchimp css file link into the <head> section.
